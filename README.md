# PhotoListMVVM

> ### Built in xcode 14.2 using Swift ~~Not SwiftUI~~
> #### Minimum deployment target 13.0.
> #### Implemented in MVVM. 
> #### Includes Unit tests and UI Tests.
> #### Swinject for DI
> #### SDWebImage for async image loading and caching

<br>

## Thirdparties - using cocoapods

- Swinject
- SDWebImage
- Quick
- Nimble
